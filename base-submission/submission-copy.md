# Pixel Shrine

## App Name

Pixel Shrine

## Tagline

Mint tiny pixel wishes

## Description

Create a tiny pixel charm with colors, symbol, and note, then save it on Base as a collectible onchain keepsake.

## Category

Creative / collectibles / personal expression

## Short Pitch

Pixel Shrine turns a small wish, symbol, and color palette into a playful onchain charm that users can mint, view, and reload by ID.

## Production Environment Variables

- `NEXT_PUBLIC_PIXEL_SHRINE_CONTRACT_ADDRESS`: deployed Pixel Shrine contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
